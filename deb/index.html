微信辅助插件一般最佳版本为6.3.16-6.3.22版本
必装依赖插件有App File Conduit 和 AppSync for Unified
和 rocketbootstrap 这三个插件！
安装插件后重启一次手机

微信降级版本可通过91助手降级
降级方法：搜索微信-点击微信图-详情页最下方
历史版本-选择版本-下载！ 一般同版本号下载最早日期！

版本降级后要去各大助手里屏蔽应用
自动更新功能以免版本又升级到最高版本！

微信分身请在pp助手里分身
教程：打开pp助手--管理--应用多开
注意事项：一般多开我们只选应用多开不选择稳定多开！
